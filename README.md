
# USAGE

```bash
# install requirements
pip install -r requirements.txt
# start geckowedriver
$ ./geckodriver -b <path_to_firefox>
# start download
$ python download.py <doujinshi_url> --save-path ./ --gecko-path /home/scarlet/geckodriver --socks-proxy <proxy_address>
```
